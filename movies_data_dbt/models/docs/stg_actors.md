# stg_actors

This table represents individual actors appearing in movies.

### What this model does
- Standardizes actor names  
- Ensures unique actor identifiers  
- Removes duplicates and inconsistent formatting  

### Columns

| Column | Description |
|--------|-------------|
| **actor_id** | Unique ID for the actor. |
| **full_name** | Full actor name as displayed publicly. |
